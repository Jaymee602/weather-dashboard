# Weather Dashboard

## Purpose 
This weather dashboard allows you to easily access the weather for today and tomorrow in a city of your choosing.

## API 
[OpenWeather One Call API](https://openweathermap.org/api/one-call-api)

## Built With
* HTML
* CSS/BootStrap
* JavaScript
